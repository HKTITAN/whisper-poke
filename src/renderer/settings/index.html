Version history
  1. v0.1.0 2026-04-21
    • Initial early-access release.
    • Global push-to-talk hotkey (Ctrl + Win by default, remappable).
    • Top-center overlay with live waveform, timer, and “Recording with Whisper Poke” caption.
    • Esc while holding cancels without sending.
    • Telegram MTProto sign-in (phone + OTP + 2FA) with keytar-backed session storage.
    • Voice notes sent as OGG/Opus to @interaction_poke_bot.
    • Subtle Web Audio SFX on start / stop / sent / cancel.
    • 1-second minimum duration so accidental taps don’t spam Poke.
    • Tray icon with Settings / khe.money / Quit.
    • About pane with version history + disclaimer.
